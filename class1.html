R Building Blocks

Key ideas

  • R has basic data structures
    • Vectors
    • Data Frames
    • Others...
  • Data structures are stored as objects
  • Functions are used to operate on objects
  • R is case sensitive

Create a Vector


Generic syntax

c(<element 1>, <element 2>)

Numeric vector

c(19, 24)

Character vector

c('Brandon', 'Carla')

Logical vector

c(FALSE, TRUE)

Store a vector as an object



Generic syntax

object.name <- <object>

Assign a numeric vector to the variable age

age <- c(19, 24)

Assign a character vector to the variable name

name <- c('Brandon', 'Carla')

Syntactically correct variable names

  • letters
  • numbers
  • dot or underline characters
  • starts with a letter
  • starts with a dot not followed by a number (e.g. ".2way" is not valid)
  • reserved words are not valid

Note: Rules for object possible names in R are flexible.

Create a Data Frame


Generic syntax

data.frame(<vector 1>, <vector 2>)

Data frame from new vectors

data.frame( c('Brandon', 'Carla'), c(19, 24))

Data frame from vectors in the workspace

data.frame( name, age)

Stored objects are added to the workspace

Objects in the workspace (a.k.a. global environment) indicate objects stored in the current memory (RAM on your local computer for local installation)


R loads data and any object created to memory. Low memory space limits the data size which can be used.

Vector arithmetic with operators

How old will our friends be in 2045


Create a numeric vector

age <- c(19, 24)

Add year difference to age vector

age + (2045 - 2018)

Manipulate objects with functions


Generic syntax

function(<object>)

Find the average age

mean(age)

Functions often have optional arguments that can be specified and default options that can be changed

Objects have a class and other properties


Find the class of an object

class(<object>)

View other properties about an object

attributes(<object>)

str(<object>)

length(<object>)

dim(<object>)

Some properties may be NULL if they have never been set or are not relevant to the object

Functions to check the type of vector

is.numeric(<vector>)

is.character(<vector>)

is.logical(<vector>)

is.factor(<vector>)

Convert objects to a different type



Generic syntax

as.something(object)

Convert a numeric vector to a character vector

as.character(<vector>)

Functions are sensitive to object class


Output may differ depending on the object class


Or the function may not work at all with some objects
