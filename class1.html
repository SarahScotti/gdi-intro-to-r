R Building Blocks

Key ideas

  • R has basic data structures
    • Vectors
    • Data Frames
    • Others...
  • Data structures are stored as objects
  • Functions are used to operate on objects
  • R is case sensitive

Create a Vector


Generic syntax

c(element 1, element 2)

Numeric vector

c(19,24)

Character vector

c('Brandon','Carla')

Logical vector

c(FALSE,TRUE)

Store a vector as an object



Generic syntax

object_name <- data

Assign a numeric vector to the variable age

age <- c(19, 24)

Assign a character vector to the variable name

name <- c('Brandon', 'Carla')

Stored object are added to the workspace

Manipulate objects with functions


Generic syntax

function( object_name )

Find the average age

mean(age)

Coerce objects to a different type



Generic syntax

as.something( object)

Create a dataframe from existing vectors

as.dataframe(name, age)

Store a dataframe as the object mydf

mydf <- as.dataframe(name, age)

Other functions: as.numeric(), as.character(), as.factor()

Objects have a type and other properties


Find the type of an object

class(object_name)

Find other properties of an object

attributes(object_name)

Functions are sensitive to class
